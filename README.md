# 💰 Тихий Финансовый Контролёр — Telegram Bot

Бот для учёта доходов и расходов в двух валютах (UZS и USD) внутри Telegram-группы.
Работает тихо: **не пишет в группу**, отчитывается только в личку.

---

## 🚀 Деплой на Railway (шаг за шагом)

### Шаг 1 — Подготовка бота
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`, задайте имя и username
3. Скопируйте **токен** — он понадобится позже

### Шаг 2 — Узнайте нужные ID
| Что нужно | Как получить |
|-----------|-------------|
| Ваш `ADMIN_ID` | Напишите [@userinfobot](https://t.me/userinfobot) |
| `ALLOWED_GROUP_ID` | Добавьте [@getidsbot](https://t.me/getidsbot) в группу — он напишет ID (отрицательное число) |

> ⚠️ ID группы всегда **отрицательный**, например `-1001234567890`

### Шаг 3 — Загрузите код на GitHub
```bash
git init
git add .
git commit -m "init finance bot"
git remote add origin https://github.com/ВАШ_НИК/finance-bot.git
git push -u origin main
```

### Шаг 4 — Создайте проект на Railway
1. Зайдите на [railway.app](https://railway.app) и залогиньтесь через GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Выберите ваш репозиторий

### Шаг 5 — Добавьте переменные окружения
В Railway: **Variables → New Variable** — добавьте по одной:

```
BOT_TOKEN          = токен от BotFather
ADMIN_ID           = ваш числовой ID
ALLOWED_GROUP_ID   = ID группы (с минусом)
ALLOWED_USER_IDS   = ID других пользователей через запятую (необязательно)
DB_PATH            = /data/finance.db
```

### Шаг 6 — Добавьте постоянный том (Volume)
Чтобы база данных не сбрасывалась при перезапуске:
1. В Railway: **Settings → Volumes → Add Volume**
2. Mount Path: `/data`
3. Нажмите **Save**

### Шаг 7 — Добавьте бота в группу
1. Добавьте бота в вашу Telegram-группу
2. Дайте боту права **читать сообщения** (Read Messages)
3. Убедитесь, что **Privacy Mode выключен**: в @BotFather → `Bot Settings → Group Privacy → Turn off`

## 📖 Синтаксис записей в группе

| Что написать | Что запишется |
|-------------|--------------|
| `+600$ 12200` | Доход: +600 USD, комментарий «12200» |
| `+28.297.000 QAHRAMON` | Доход: +28 297 000 UZS, комментарий «QAHRAMON» |
| `-150000 аренда` | Расход: -150 000 UZS, комментарий «аренда» |
| `50000` | Расход: -50 000 UZS (без знака = расход) |
| `-50$` | Расход: -50 USD |

## 🔒 Безопасность
- Бот обрабатывает сообщения **только из вашей группы** (`ALLOWED_GROUP_ID`)
- Баланс показывает **только разрешённым пользователям** (`ADMIN_ID` + `ALLOWED_USER_IDS`)
- В группе бот **никогда не пишет** — только слушает
